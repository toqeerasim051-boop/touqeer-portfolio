# 💼 Touqeer Asim - Full Stack Developer Portfolio

A modern, responsive portfolio website showcasing my work as a Full Stack Web Developer. Built with PHP, MySQL, Bootstrap, and JavaScript.

## 🚀 Live Demo

🔗 [View Live Portfolio](https://touqeer-portfolio.infinityfree.me/)

## ✨ Features

### Frontend
- 🎨 Modern design with dark/light theme toggle
- 📱 Fully responsive on all devices
- 🧩 Smooth scroll animations
- 🖼️ Dynamic project cards with images
- 📧 Contact form with validation
- 🔗 Social media integration (GitHub, LinkedIn, WhatsApp, Facebook, Email)

### Admin Panel
- 🔐 Secure admin authentication
- 📊 Dashboard overview
- ➕ Add/Edit/Delete projects
- 🏷️ Project type & tech stack management
- 👁️ Publish/Unpublish projects
- 📨 View and manage contact messages
- 🖼️ Profile photo upload
- 🎨 Dark/Light theme preference

### Technical
- 🗄️ MySQL database for dynamic content
- 📁 File upload system
- 🔄 CSRF protection
- 📱 Contact form with IP logging
- 🎯 Session-based authentication

## 🛠️ Technologies

| Category | Technologies |
|----------|-------------|
| Frontend | HTML5, CSS3, JavaScript, Bootstrap 5 |
| Backend | PHP 7.4+ |
| Database | MySQL |
| Icons | Bootstrap Icons |
| Fonts | Google Fonts (Raleway, Poppins) |
